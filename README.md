# Gotlee-Neue

Gotlee-Neue is an open-source typeface designed for modern digital interfaces, branding, and editorial use.  
It balances clarity and character, making it suitable for both display and functional typography contexts.

## Styles
- Gotlee-Neue Medium
- Gotlee-Neue Regular  
*(Additional weights and styles may be added in future releases.)*

## Usage
You are free to use Gotlee-Neue for:

- Personal and commercial projects
- Websites, apps, and UI design
- Branding, marketing, and editorial layouts
- Print and digital media

You may also modify and redistribute the font under the terms of the license.

## Installation
1. Download the font files from the `fonts/` directory.
2. Install the `.otf` files on your system.
3. Use the font in your design or development tools as you would any other font.
